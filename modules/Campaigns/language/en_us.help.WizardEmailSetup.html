Email Setup for Campaigns

Use this page to specify email settings and a bounce-handling inbox for bounced campaign emails .

The Email Setup for Campaigns page displays the following information:

  • From Name. Enter the name of the sender.
  • Mail Transfer Agent. From the drop-down list, select either sendmail or SMTP as the protocol.
  • From Address. Enter the email address of the sender. If you select SMTP, enter the server name, port number, password, and username; Select the Use SMTP authentication box.
  • Number of emails sent per batch. Enter the maximum number of emails to send per batch.
  • Location of campaign tracking files. To specify where the campaign tracking files should be saved, choose Allow Select and enter the path to the location in the field below. To save it to the default location, choose Auto-Create.
  • Keep copies of campaign messages. To keep copies of the campaign messages, select Yes. If not, select No.
Click Next to create a new mailbox.

New Mail Box

  • Create New Mailbox. Select this box to create a new mailbox.
  • Mailbox Name. Enter a name for the mailbox.
  • From Address. Enter the email address of the sender.
  • Mail Server Address. Enter the address of your mail server.
  • Mail Server Protocol. From the drop-down list, select either IMAP or POP3.
  • Mail Server Port. Enter the server port number.
  • Password. Enter the sender's password.
  • Monitored Folder. By default, this field displays INBOX as the monitored folder. Click Next to view a summary of your email settings.
