Marketing Email

Use this page to create an email marketing record that contains information such as the campaign target list, the campaign email template, and the inbox to handle emails that bounce back to you.

Enter the following information:

  • Name. Enter a name for the record.
  • Status. From the drop-down list, select Active if the record is in use; select Inactive if the record is not in use. Only active records will be queued for campaign launch.
  • Use Mailbox. From the drop-down list, select a mailbox that is set up for BounceHandling.
  • From Name. Enter the name of the organization or person that is sending out the email.
  • Start Date and Time. Click the calendar icon and select a date to send out the email. Enter the time in the adjoining field.
  • Email Template. From the drop-down list, select an existing email template; to create a new template or edit an existing one, click the adjoining Create or Edit link respectively.
  • Send This Message To. Select one or more target lists from the box below; to send it to all the target lists in the box, select the All Target Lists in the Campaign box.
  • Click Next to view the next step in the process; click Back to go back to the previous step; click Cancel to exit the page without saving your changes.

Send Email

  • Finish: Select this option to save the campaign information and view the summary.
  • Send Marketing Email as Test: Select this option to send a test email to the Test subscription list.
  • Schedule Email: Select this option to add the campaign email to the email queue.
