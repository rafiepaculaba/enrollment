Campaigns

Use the Campaigns module to create and manage campaigns for your organization. You can create the many types of marketing campaigns such as Email, Newsletter, Mail, Web, Radio, Television, Print, and Telesales.

The Campaigns Home page displays the following information:

  • A Search sub-panel where you can enter the campaign name to search for a specific campaign.
    • To search only for campaigns assigned to you, select Only my items.
    • To perform an advanced search using additional fields, click the Advanced Search tab.
    • To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A Shortcuts menu that lists the following options:
    • Campaign Wizard. Click this option to use the Campaign Wizard for step-by-step guidance while creating a campaign.
    • Create Campaign (Classic) . Click this option to create a new campaign. Alternatively, enter the required information in the New Campaign quick form. You can specify additional information after you save it.
    • Campaigns. Click this option to navigate back to the Campaigns Home page from a campaign's detail page.
    • Newsletters. Click this option to view a list of existing newsletters.
    • Create Target List. Click this option to create a list of targets for a campaign.
    • Target Lists. Click this option to view existing target lists.
    • Create Target. Click this option to create a campaign target.
    • Targets. Click this option to view a list of existing targets.
    • Create Email Template. Click this option to create a template for a campaign email.
    • Email Templates. Click this option to view a list of existing campaign email templates.
    • Import Targets. Click this option to import target information from an external source.
    • Campaign Wizard. Click this option to create a campaign.
    • Email Setup. Click this option to specify email settings for the campaign. You must be an administrator to perform this task.
    • Diagnostic. Click this option to run campaign diagnostics to determine whether all the administrative tasks have been completed to ensure that the campaign can be successfully launched.
    • Create Lead Form. Click this option to create a Web-to-lead form.
  • A list of existing campaigns along with related information such as names and phone numbers.
    • To view a campaign's details, click its name or the View icon to the right of the target information in the list
    • To export one or more campaigns, select them from the Campaign list, click the Export link, and choose Selected Records. To export all the campaigns listed on the page or the entire list, select Current Page and Entire List respectively.
    • To update or delete multiple campaigns, select them from the list, and use the Mass Update sub-panel.
  • A New Campaign quick form that you can use to enter the required information to create a new campaign. You can enter additional information after you create the campaign.
