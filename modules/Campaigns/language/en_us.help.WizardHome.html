Campaign Wizard

The Campaign Wizard takes you through the process of creating a campaign and performing related tasks.

To create a campaign, enter the following information:

  • Newsletter. Select this option to create a newsletter campaign.
  • Email.Select this option to create an email campaign.
  • Non-email based Campaign.Select this option to create other types of campaign such as a telesales campaign.
  • Start. Click this button to view the next page.

    • After you create a campaign, the campaign summary page displays. You can edit the campaign information, view the campaign status, and view the campaign's return on investment (ROI) from this page.
